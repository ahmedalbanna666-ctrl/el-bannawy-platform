# El-bannawy Platform
## Ask El-bannawy AI UI Specification

Version: 1.0.0

---

# Purpose

Defines the complete user interface for the Ask El-bannawy AI experience.

The AI assistant is designed to teach, guide and support students throughout their learning journey.

---

# Design Philosophy

The AI experience must feel:

- Helpful
- Fast
- Friendly
- Educational
- Non-intimidating

Students should feel they are learning with a teacher, not chatting with a robot.
