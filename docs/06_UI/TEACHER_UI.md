# Purpose

Defines the complete Teacher Dashboard experience.

The dashboard allows teachers to monitor students, manage educational content, configure lesson settings, timeline events, completion rules and analyze classroom performance.

# Lesson Creation Flow

Create Lesson

↓

Provide YouTube Unlisted URLs (one or more per lesson)

↓

Upload Microsoft Word Document

↓

System Converts Word Document into Activities

↓

Configure Lesson Settings (Sequential / Any-Order)

↓

Configure Interactive Timeline (per video)

↓

Configure Required / Optional Activities (per video)

↓

Configure Completion & Unlock Rules

↓

Publish Lesson

Teachers do not manually create activities inside the dashboard.

Lesson creation requires only YouTube URLs and a Word document.

The platform does not store video files.
