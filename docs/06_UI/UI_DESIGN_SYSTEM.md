# El-bannawy Platform
## UI Design System

Version: 1.0.0

---

# Purpose

This document defines the official Design System of the El-bannawy Platform.

Every screen, component, animation and interaction must follow these standards.

No page may introduce new design patterns outside this document without approval.

---

# Design Philosophy

The El-bannawy Platform is:

- Modern
- Premium
- Educational
- Gamified
- Clean
- Fast
- Accessible

It is NOT:

- Childish
- Traditional LMS
- Corporate ERP
- Flat without depth
- Overloaded with decorations

---

# Design Principles

1. Mobile First

All interfaces are designed for mobile first.

Desktop layouts are enhancements.

---

2. Consistency

Every page follows the same spacing, typography and component rules.

---

3. Simplicity

Every screen must answer:

What should the student do next?

Nothing should distract from learning.

---

4. Performance

Animations are lightweight.

No unnecessary rendering.

60 FPS target.

---

5. Accessibility

Support

- Keyboard Navigation
- Screen Readers
- High Contrast
- Focus States

WCAG AA minimum.

# Dark Theme

Primary Experience

Dark Mode is the default platform appearance.

Dark backgrounds use subtle gradients.

Avoid pure black.

---

# Light Theme

Optimized for readability.

Minimal shadows.

Soft backgrounds.

---

# Glassmorphism

Used only for:

- Hero Cards
- Dashboard Cards
- AI Cards
- Live Cards

Do not overuse glass effects.
