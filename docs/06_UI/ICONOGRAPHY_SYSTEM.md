# ICONOGRAPHY_SYSTEM.md

# El-bannawy Platform
## Iconography System

Version: 1.0.0

---

# Purpose

This document defines the official icon system used throughout the El-bannawy Platform.

Icons are visual shortcuts that improve usability and reduce cognitive load.

Every icon must communicate meaning instantly.

---

# Design Philosophy

Icons should be:

- Modern
- Minimal
- Professional
- Educational
- Consistent
- Easy to Recognize

Icons must support both Dark Mode and Light Mode.

# Empty States

Use friendly illustrations with supporting icons.

Never use icons alone.

Always include:

- Title

- Description

- Primary Action

---

# Accessibility

Icons must never be the only source of information.

Every interactive icon requires:

- Accessible Label

- Tooltip

- Keyboard Focus

---

# Animation

Allowed

Fade

Rotate

Scale

Glow

Forbidden

Bounce

Flash

Excessive Motion

---

# Performance

Icons should be:

SVG Only

Tree Shaken

Lazy Loaded when appropriate

No PNG icons.

# Acceptance Criteria

✓ SVG Icons

✓ Accessible

✓ Responsive

✓ Dark Mode Compatible

✓ Consistent Stroke Width

✓ Lucide First

✓ Mobile Optimized

---

# Final Rule

Icons are part of the platform language.

Every icon must communicate a single clear action or concept and remain visually consistent across the entire El-bannawy Platform.
