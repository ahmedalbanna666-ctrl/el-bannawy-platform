# AI_MEMORY_SYSTEM.md

# El-bannawy Platform
## AI Memory System

Version: 1.0.0

---

# Purpose

This document defines how the AI remembers information during conversations while protecting user privacy.

The AI Memory System improves personalization without permanently storing unnecessary information.

---

# Memory Types

Session Memory

Short-Term Memory

Long-Term Memory

Educational Memory

System Memory

---

# Session Memory

Lifetime

Current Conversation

Stores

- Current Lesson
- Current Question
- Current Context

Deleted after session ends.

---

# Short-Term Memory

Lifetime

7 Days

Stores

- Recent Questions
- Recent Lessons
- Recent Mistakes
- Recent AI Recommendations

---

# Long-Term Memory

Stores

- Preferred Language
- Learning Style
- Weak Skills
- Strong Skills
- Preferred Explanation Style

---

# Educational Memory

Tracks

Completed Lessons

Vocabulary Progress

Grammar Progress

Homework History

Quiz History

Review History

---

# Memory Rules

Never store

Passwords

Payment Data

Private Messages

Sensitive Information

---

# Memory Retrieval

Student Request

↓

Load Educational Profile

↓

Load Current Lesson

↓

Load Recent Context

↓

Merge with RAG Context

↓

Generate Response

---

# Memory Update

Occurs after

Lesson Completed

Quiz Finished

Homework Submitted

AI Conversation Finished

# Privacy

Reset AI Memory
