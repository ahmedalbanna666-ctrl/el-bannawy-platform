# El-bannawy Platform
## Secretary Dashboard Requirements

Version: 1.0.0

---

# Purpose

The Secretary Dashboard is the operational management center responsible for handling daily administrative tasks related to students, subscriptions, payments and communication.

The secretary is responsible for administrative operations only.

The secretary has NO authority over educational content.

---

# Objectives

The Secretary Dashboard must:

- Register students.
- Manage subscriptions.
- Verify payments.
- Activate educational content.
- Schedule live classes.
- Communicate with parents.
- Generate administrative reports.

# Student Registration

Secretaries can:

- Register New Students
- Edit Student Information
- Activate Student Accounts
- Suspend Accounts
- Reset Passwords
- Upload Student Photos

Required Information

- Full Name
- Mobile Number
- Parent Mobile
- Educational Stage
- Grade

---

# Subscription Management

Secretaries can:

- Create Subscription
- Renew Subscription
- Upgrade Subscription
- Expire Subscription
- Cancel Subscription

Subscription Types

- Lesson
- Unit
- Full Course
- Premium Package

---

# Payment Management

Secretaries can:

- Verify Payments
- Confirm Transactions
- Record Offline Payments
- View Payment History
- Print Receipts

Supported Methods

- Paymob
- Fawry
- Instapay

---

# Coin Management

Secretaries may:

- Verify Coin Purchases
- View Wallet
- Confirm Payment

Secretaries cannot:

- Create Coins
- Remove Coins
- Modify Coin Balance

Only administrators can perform those actions.
