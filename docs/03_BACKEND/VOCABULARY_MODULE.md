# VOCABULARY_MODULE.md

# El-bannawy Platform
## Vocabulary Module Requirements

Version: 1.0.0

---

# Purpose

The Vocabulary Module is responsible for teaching, reinforcing, and assessing the vocabulary associated with each lesson.

Vocabulary is considered a core learning component and is independent from the lesson video.

Students should be able to revisit vocabulary at any time.

---

# Objectives

The Vocabulary Module must:

- Teach new vocabulary.
- Improve pronunciation.
- Improve spelling.
- Improve meaning recognition.
- Improve sentence usage.
- Prepare students for homework and quizzes.

# Vocabulary Structure

Each Lesson owns exactly one Vocabulary Collection.

A Vocabulary Collection contains multiple Vocabulary Items.

---

# Vocabulary Item

Every Vocabulary Item must contain:

- Word
- Pronunciation
- Arabic Meaning
- English Definition
- Example Sentence
- Lesson Reference
- Unit Reference

Optional

- Image
- Audio Pronunciation
- Video Explanation
- Synonyms
- Antonyms
- Notes

---

# Word Card

Each vocabulary card displays:

- Word
- Pronunciation Button
- Meaning
- Example
- Favorite Button
- Completed Indicator

---

# Pronunciation

Students can replay pronunciation unlimited times.

Audio playback should start instantly.

---

# Vocabulary Navigation

Students may:

- Next Word
- Previous Word
- Replay Audio
- Mark as Learned
- Save Favorite

---

# Mark as Learned

Students may mark a word as learned.

This affects progress only.

It does not unlock lessons.

---

# Favorites

Students may add vocabulary to Favorites.

Favorite words appear inside the student's personal vocabulary collection.

---

# Search

Students may search by:

- Word
- Meaning

Search should be instant.

---

# Filters

Supported Filters

- All Words
- Learned
- Not Learned
- Favorites

Students can practice vocabulary using:

- Flashcards
